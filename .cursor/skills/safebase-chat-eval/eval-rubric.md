# 对话质量评分参考

## 用户模拟（1–5）

| 分 | 标准 |
|----|------|
| 5 | 纯打字、回应上轮、新人设细节、无舞台指示 |
| 3 | 大体自然，偶有文艺腔或轻微重复 |
| 1 | 预写台词感、括号动作、与伙伴回复脱节 |

## 疗愈伙伴（1–5）

| 分 | 标准 |
|----|------|
| 5 | 反映+串联+适度假设+开放收尾；引用记忆有据；口语 |
| 3 | 温暖但泛化；记忆引用浅；略长或略模板 |
| 1 | 报告体/清单体；幻觉记忆；括号动作；说教 |

## 常见修复路径

| 问题 | 改哪里 |
|------|--------|
| 用户像配音 | `seed-trauma-users.mjs` → `CHAT_STYLE_RULES` |
| 伙伴像咨询师报告 | `prompt.ts` → 输出格式、篇幅、避免列表 |
| 伙伴不记得日记 | RAG/`match_diaries`；检查 embedding 与 cron |
| 伙伴重复问已答问题 | `prompt.ts` [近期对话] 指引；检查 `short_ctx` |
